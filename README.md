# PhenoVCS

Rust workspace providing VCS (Version Control System) primitives for the Phenotype ecosystem.

## Key Features

- **pheno-vcs-core** - VCS primitives

## Quick Start

```bash
cargo build
cargo test
cargo clippy --workspace -- -D warnings
```
